Aide

Comment enregistrer un nouveau client ?

Dans la page d’accueil, après s’être connecté avec son identifiant (numéro interne), vous serez redirigé vers une page qui contient un formulaire avec les champs suivant :

  • Nom et Prénom
  • Numéro de téléphone et E-mail
  • TeamViewerID et TeamViewerPW
  • Echéance et rendez-vous
  • Objet
  • Notification personnalisée
  • Bouton enregistrer
  • Bouton historique
  • Bouton aide
  • lien retour à l’authentification

Contraintes pour les champs à remplir

Il faut remplir au moins un des deux champs (nom ou prénom), et qu’ils ne dépassent pas 50 caractères chacun.

Il faut remplir au moins un des deux champs (numéro de téléphone ou E-mail), et respecter les formats standards européens.

Les champs TeamViewerID et TeamViewerPW sont facultatifs et permettent la prise en main de l’ordinateur client par un utilisateur pour l’assistance technique.

Le champ échéance et rendez-vous est un champ text.

De même que pour le champ objet mais avec une capacité plus importante.

La case à cocher pour le champ “Notification personnalisée” est cochée par défaut;

si ce champ est vide, le contenu du formulaire sera envoyé par E-mail à tous les chargés clients.Sinon, on peut le remplir avec les collaborateur de son choix.

Si la case à cocher est décoché, il n’y aura pas d’envoi.  

Boutons proposés

Le bouton “Enregistrer” effectue l’enregistrement du formulaire, sauvegarde les éléments dans la base de donnée et envoi un E-mail aux chargés client si la case est cochée.

Le bouton “Historique” permet de lister tous les clients de la base.Chaque client contient les informations du formulaire et le dernier chargé client qui l’a enregistrer.

Le bouton “Rechercher” permet de saisir une occurrence comprise dans le nom ou dans le prénom du client.En cas d’homonyme une liste sera affichée.

Le bouton “Aide” renvoi à ce document.

Un lien est proposé pour un retour à l’authentification.Dans le cas où un autre chargé client voudrait enregistrer des données client.

Tables de la base de donnée

La base de donnée est constituée de deux tables:

  • Table client
  • Table chargé client

La table client comprend les champs:

  • ID (identifiant)
  • Nom
  • Prénom
  • Tel
  • Mail
  • TeamViewerID
  • TeamViewerPW
  • RDV
  • Objet
  • cc_nom
  • cc_prenom
  • cc_telint (téléphone interne)
  • cc_portable
  • cc_email
  • Date

La table chargé client comprend les champs:

  • cc_id
  • cc_nom
  • cc_prenom
  • cc_telint
  • cc_portable
  • cc_email
